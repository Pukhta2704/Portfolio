1.SocioRama

Click here to See More

This is a full stack app. This app is an Instagram like social media app where the user can follow unfollow people, like unlike posts, add delete comments and the user can also create and delete posts. I've used React with Redux for state management, materialize css and vanilla css for the front end, NodeJs, ExpressJs and Mongodb for the backend. For authentication I've used BcryptJs and for authorization I've used JSON web tokens. I've also set up pagination in the app so the user can see posts ininfitely. I've set up the app so that when you visit it from this website, you will be automatically logged in to an existing account. You can also sign up later.
